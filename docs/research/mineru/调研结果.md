1. MinerU 云端服务 — 调研结果
                                                                                                                 
  结论：MinerU 确实提供云端 API 服务，有两种模式：
                                                                                                                 
  ┌──────────┬───────────────────────┬──────────────────────────────┐                                            
  │  对比项  │ Flash Extract（免费） │ Precision Extract（需Token） │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 认证     │ 无需Token，IP限流     │ 需要API Token                │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 文件大小 │ ≤ 10 MB               │ ≤ 200 MB                     │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 页数限制 │ ≤ 20 页               │ ≤ 600 页                     │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 模型版本 │ 固定轻量模型          │ pipeline / vlm(推荐) / html  │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 输出格式 │ 仅 Markdown           │ MD, JSON, DOCX, HTML, LaTeX  │
  ├──────────┼───────────────────────┼──────────────────────────────┤
  │ 批量支持 │ 单文件                │ 支持（≤200文件）             │
  └──────────┴───────────────────────┴──────────────────────────────┘

  2. MinerU 云端 API 可配置参数

  ┌───────────────┬──────────┬──────────────────────────────────────────┐
  │     参数      │  默认值  │                   说明                   │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ model         │ 自动推断 │ "vlm" / "pipeline" / "html"              │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ ocr           │ 关闭     │ 是否启用 OCR（扫描件必须开启）           │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ formula       │ 开启     │ 公式识别                                 │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ table         │ 开启     │ 表格识别                                 │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ language      │ "ch"     │ 文档语言（ch/en/korean/japan 等 109 种） │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ pages         │ 全文     │ 页码范围，如 "1-20"                      │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ extra_formats │ 无       │ 额外导出格式：docx / html / latex        │
  ├───────────────┼──────────┼──────────────────────────────────────────┤
  │ timeout       │ 300s     │ 轮询超时时间                             │
  └───────────────┴──────────┴──────────────────────────────────────────┘
