## 二、左侧导航规范

### 尺寸与背景

- 宽度：`220px`，固定不滚动
- 背景：`#FFFFFF`
- 右侧边框：`1px solid #D9E1E8`

### Logo 区域

- 高度：`56px`（与顶部栏对齐）
- 内边距：`0 16px`
- 产品名：`16px 600 #102A43`

### 导航项规范

- 高度：`36px`
- 内边距：`8px 12px`
- 图标：Lucide `16px`，与文字间距 `8px`
- 文字：`13px 500 #486581`

### 导航项状态

| 状态 | 背景 | 文字颜色 | 左侧指示条 |
|------|------|----------|------------|
| 默认 | 透明 | `#486581` | 无 |
| 悬停 | `#F0F4F8` | `#102A43` | 无，150ms |
| 激活 | `#EBF4FF` | `#0F3D5E` | `3px solid #0F3D5E` |

### 分组标题

- 字号：`11px 500`，大写字母
- 颜色：`#7B8794`
- 字间距：`0.08em`
- 内边距：`16px 12px 4px`

### 底部配置项

- 与主导航用 `1px #D9E1E8` 分隔线隔开
- 系统配置单独分组，视觉权重低于主导航

## 三、顶部上下文栏规范

- 高度：`56px`
- 背景：`#FCFDFE`
- 底部边框：`1px solid #D9E1E8`
- 左侧：当前页面面包屑（`13px #7B8794`，当前页加深）
- 中间：当前知识库选择器（全局上下文切换，下拉）
- 右侧：全局搜索图标（`⌘K` tooltip）+ 通知图标

---

## 四、主工作区规范

- 最大宽度：`1440px`
- 内边距：`24px 32px`
- 背景：`#F5F7FA`
- 垂直滚动，不出现水平滚动

### 页面头部规范

每个页面顶部统一结构：

```
页面标题（h1，30px 700 #102A43）
副标题或描述（14px #486581，可选）
右侧操作按钮区（与标题垂直居中对齐）
```

- 头部底部间距：`24px`

## 五、右侧详情抽屉规范

适用于：文档详情、切块详情、引用详情、错误详情

- 宽度：`400px`（文档/切块详情）、`360px`（引用详情）
- 从右侧滑入，`200ms ease-out`
- 背景：`#FFFFFF`
- 左侧边框：`1px solid #D9E1E8`
- 阴影：`0 12px 28px rgba(16,42,67,0.12)`
- 顶部：标题 + 关闭按钮（X，Lucide `20px`）
- 内容区：`padding: 20px`，垂直滚动

## 六、栅格系统

- 主栅格：`12 列`
- 列间距：`24px`

| 页面 | 分栏 | 说明 |

## 七、响应式断点

| 断点 | 宽度 | 行为 |
|------|------|------|
| Mobile | `375px+` | 左侧导航收起，底部 Tab 导航 |
| Tablet | `768px+` | 左侧导航收起为图标模式（48px） |
| Desktop | `1024px+` | 完整双列布局，左侧导航展开 |
| Wide | `1440px+` | 三段式分析工作台 |

### Mobile 降级规则

- 左侧导航 → 底部 Tab 导航（总览/知识库/入库/问答/更多）
- 三栏布局 → 垂直堆叠，次要面板默认折叠
- 右侧抽屉 → 全屏底部面板（bottom sheet）
- 表格 → 卡片列表

### Tablet 降级规则

- 左侧导航收起为图标模式，悬停展开
- 右侧抽屉改为覆盖式（不推挤主内容）
- 三栏合并为两栏

## 八、信息优先级规则

页面中信息优先级顺序：

1. 当前状态（系统/任务/链路状态）
2. 核心结果（答案、切块数、评分）
3. 关键异常（失败、降级、告警）
4. 可执行动作（主操作按钮）
5. 证据细节（切块内容、引用来源）
6. 参数与次要元信息（配置、ID、时间戳）
