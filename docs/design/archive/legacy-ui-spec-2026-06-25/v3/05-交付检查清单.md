# 05 交付检查清单

---

## 一、设计令牌合规

- [ ] 所有颜色值来自 `01-Design-Tokens.md`，无硬编码色值
- [ ] 字号、字重、行高符合字体系统定义
- [ ] 间距值使用 `--space-*` 令牌，无随意像素值
- [ ] 圆角使用 `--radius-*` 令牌
- [ ] 阴影使用 `--shadow-*` 令牌
- [ ] 动效时长不超过 `300ms`，使用规定缓动函数

---

## 二、组件规范合规

- [ ] 按钮变体使用正确（每页 Primary 按钮不超过 1 个）
- [ ] 状态徽标颜色与语义匹配（success/warning/danger/pending/running）
- [ ] 表格表头使用 `12px 500 #7B8794` 大写样式
- [ ] 数字/分数/ID 使用 `--font-mono`（IBM Plex Mono）
- [ ] 图标统一使用 Lucide，无混用其他图标库
- [ ] 纯图标按钮均有 `aria-label`
- [ ] 空状态使用规范组件，非自定义文字

---

## 三、布局规范合规

- [ ] 左侧导航宽度 `220px`，顶部栏高度 `56px`
- [ ] 主工作区最大宽度 `1440px`，内边距 `24px 32px`
- [ ] 页面头部结构统一（h1 标题 + 可选描述 + 右侧操作区）
- [ ] 右侧抽屉宽度符合规范（400px 或 360px）
- [ ] 三栏布局比例符合对应页面规范

---

## 四、响应式

- [ ] Desktop（1024px+）：完整双列/三列布局正常
- [ ] Tablet（768px+）：左侧导航收起为图标模式（48px）
- [ ] Mobile（375px+）：底部 Tab 导航，三栏变垂直堆叠
- [ ] 无水平滚动条（任何断点）
- [ ] 右侧抽屉在 Mobile 降级为底部面板（bottom sheet）

---

## 五、交互状态完整性

- [ ] 所有按钮有：默认、悬停、聚焦、禁用、加载中 五态
- [ ] 所有输入框有：默认、悬停、聚焦、错误、禁用 五态
- [ ] 所有表格行有：默认、悬停、选中 三态
- [ ] 所有导航项有：默认、悬停、激活 三态
- [ ] 加载中状态：骨架屏或 Loader 图标，无空白闪烁
- [ ] 错误状态：有明确错误提示，非静默失败

---

## 六、流水线页面专项

- [ ] 10 个阶段节点全部有对应状态（待处理/进行中/完成/失败）
- [ ] 进行中阶段有脉冲动画
- [ ] 失败阶段有错误摘要，可展开查看日志
- [ ] 阶段颜色与 Design Tokens 流水线阶段色一致
- [ ] 实时日志区域有自动滚动到底部行为
- [ ] 任务队列支持多任务切换，状态独立

---

## 七、知识问答页面专项

- [ ] 三栏布局在 Wide（1440px+）下正常展示
- [ ] 召回通道标签页切换流畅，数据不重置
- [ ] 相似度分数颜色编码正确（≥0.9 绿/≥0.7 橙/<0.7 红）
- [ ] 切块内容超出 4 行有折叠/展开交互
- [ ] 答案引用来源可点击跳转到对应切块
- [ ] 无法回答时有明确提示，非空白

---

## 八、可访问性（Accessibility）

- [ ] 色彩对比度：正文文字 ≥ 4.5:1，大文字 ≥ 3:1（WCAG AA）
- [ ] 所有交互元素可通过键盘操作（Tab 顺序合理）
- [ ] 焦点状态可见（`2px solid #0F3D5E` outline）
- [ ] 图标按钮有 `aria-label`
- [ ] 状态变化有 `aria-live` 区域通知（流水线状态更新）
- [ ] 表格有 `<caption>` 或 `aria-label`
- [ ] 弹窗有焦点陷阱（Tab 不离开弹窗）
- [ ] 颜色不是传达信息的唯一手段（状态同时有图标+文字）

> **注意**：完整无障碍验证需配合屏幕阅读器（NVDA/VoiceOver）手动测试，本清单为基础自查项。

---

## 九、性能

- [ ] 首屏关键内容（LCP）在 `2.5s` 内可见
- [ ] 骨架屏在数据加载前立即显示，无白屏
- [ ] 表格/列表超过 100 条时使用虚拟滚动
- [ ] 图片资源有 `width`/`height` 属性，防止布局偏移（CLS）
- [ ] 动效不在低性能设备上卡顿（`prefers-reduced-motion` 媒体查询支持）

---

## 十、数据与边界情况

- [ ] 知识库名称超长时截断并有 Tooltip 显示完整名称
- [ ] 文件名超长时截断（中间省略，保留扩展名）
- [ ] 数字超大时有千分位格式（如 `12,480`）
- [ ] 时间显示：7天内用相对时间，超过7天用绝对日期
- [ ] 空列表/空知识库有空状态组件，非空白区域
- [ ] 网络断开时有全局提示，恢复后自动重连

---

## 十一、代码交付规范

- [ ] CSS 变量名与 Design Tokens 文件一致
- [ ] 组件命名语义化，与规范文档中的组件名对应
- [ ] 无内联样式（`style=""` 属性），全部使用 CSS 类
- [ ] 响应式使用 CSS 媒体查询，非 JS 控制
- [ ] 动效使用 CSS transition/animation，非 JS 定时器
