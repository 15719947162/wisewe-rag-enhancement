# 07 页面线框与链路映射

## 1. 总览页线框

### 区块结构

1. 页面头部
2. 顶层指标卡
3. 离线链路健康区
4. 在线链路健康区
5. 异常与待处理任务区

### 关键内容

- 总 KB 数
- 总文档数
- 最近 24h 建库任务
- 在线问答质量
- 失败 / 降级告警

## 2. 知识库页线框

### 区块结构

1. KB 列表
2. 文档列表
3. 文档详情
4. chunk 抽屉

### 链路映射

- `knowledge_bases`
- `documents`
- `chunks`

## 3. 离线建库页线框

### 页面结构

1. 顶部任务头
2. 左侧任务队列
3. 中部阶段工作区
4. 右侧详情抽屉

### 阶段映射

- 文档接入
- OSS 上传
- 云解析
- 内容规范化
- 清洗
- 切片
- 关联补链
- 质量过滤
- 向量化
- 索引落库

### 阶段对应展示建议

#### 解析

- block 数量
- block 类型分布
- 原始样本预览

#### 清洗

- 删除数量
- 删除原因
- before / after 对比

#### 切片

- 策略切换
- chunk 数量
- parent / child / enhanced 结构

#### 质量过滤

- discarded count
- score 分布
- 过滤原因

#### 向量化与导出

- embedding 数量
- CSV 路径
- pgvector 写入结果

## 4. 在线问答页线框

### 页面结构

1. 左侧 Query Config
2. 中部 Recall + Rerank + Context
3. 右侧 Answer + Citation + Score

### 链路映射

- query input
- HybridRetriever
- coarse filter
- ParentChildReranker
- context window
- RAGGenerator
- citation align
- RAGScorer

### 关键联动

- 点击答案中的 citation -> 高亮 citation card
- 点击 citation card -> 打开 chunk 抽屉
- 点击 chunk -> 定位 source / page

## 5. 评测分析页线框

### 页面结构

1. 分数总览
2. 趋势图
3. 低分样本列表
4. 单样本诊断抽屉

### 链路映射

- relevance_score
- faithfulness_score
- llm_score
- cannot_answer

### 关键视图

- relevance vs faithfulness 散点图
- 失败样本表格
- 原问题 / 最终答案 / 引用 / 评分 明细

## 6. 系统配置页线框

### 页面结构

1. 全局模型配置
2. 清洗配置
3. 切片配置
4. 召回配置
5. 数据库配置

### 链路映射

- `LLM_API_KEY`
- `LLM_BASE_URL`
- `RAG_LLM_*`
- `LLM_CLEANER_MODEL`
- `top_k`
- `min_score`
- `strategy`

## 7. 页面之间的跳转关系

- 总览 -> 失败任务 -> 离线建库详情
- 总览 -> 低分问答 -> 在线问答详情
- 知识库 -> 文档 -> chunk
- 在线问答 -> citation -> chunk -> source page
- 评测分析 -> 样本详情 -> 在线问答工作台

## 8. 实现优先级建议

第一阶段建议优先做：

1. 总览
2. 离线建库
3. 在线问答

第二阶段补：

4. 知识库
5. 评测分析
6. 系统配置
