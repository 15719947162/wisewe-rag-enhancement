# 03 核心组件规范

---

## 一、按钮（Button）

### 变体

| 变体 | 背景 | 文字 | 边框 | 用途 |
|------|------|------|------|------|
| Primary | `#0F3D5E` | `#FFFFFF` | 无 | 页面主操作（每页最多1个） |
| Secondary | `#FFFFFF` | `#0F3D5E` | `1px #D9E1E8` | 次要操作 |
| Ghost | 透明 | `#486581` | 无 | 工具栏操作、内联操作 |
| Danger | `#C2410C` | `#FFFFFF` | 无 | 删除、不可逆操作 |
| Danger Ghost | 透明 | `#C2410C` | 无 | 低强调危险操作 |

### 尺寸

| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| sm | `28px` | `0 10px` | `12px 500` |
| md（默认） | `34px` | `0 14px` | `13px 500` |
| lg | `40px` | `0 18px` | `14px 500` |

### 状态规则

- 悬停：Primary 变 `#14557F`，Secondary/Ghost 背景变 `#F0F4F8`，`150ms`
- 聚焦：`2px solid #0F3D5E` outline，offset `2px`
- 禁用：`opacity: 0.4`，`cursor: not-allowed`
- 加载中：左侧显示 `16px` Lucide `Loader2` 旋转图标，文字不变

### 图标按钮

- 纯图标：`28px × 28px`（sm）/ `34px × 34px`（md），必须有 `aria-label`
- 图标+文字：图标 `16px`，与文字间距 `6px`

## 二、徽标（Badge）

### 状态徽标

| 状态 | 背景 | 文字 | 用途 |
|------|------|------|------|
| success | `#F0FDF4` | `#1F9D55` | 完成、有效 |
| warning | `#FFFBEB` | `#D97706` | 部分问题、降级 |
| danger | `#FEF2F2` | `#C2410C` | 失败、阻塞 |
| info | `#EBF4FF` | `#2563EB` | 信息提示 |
| pending | `#F1F5F9` | `#94A3B8` | 排队中 |
| running | `#EBF8FF` | `#0F9F8C` | 处理中 |
| degraded | `#FFFBEB` | `#B45309` | 已完成但降级 |

### 规格

- 圆角：`--radius-pill`（999px）
- 内边距：`2px 8px`
- 字号：`12px 500`
- 运行中状态：左侧附 `6px` 脉冲圆点（同色系）

### 数字徽标

- 用于导航项未读数、告警计数
- 尺寸：`16px × 16px`（≤9）/ 自适应宽度（≥10）
- 背景：`#C2410C`，文字：`#FFFFFF`，字号：`10px 600`

---

## 三、输入框（Input）

### 规格

- 高度：`34px`（md）/ `28px`（sm）
- 内边距：`0 12px`
- 圆角：`--radius-sm`（8px）
- 边框：`1px solid #D9E1E8`
- 背景：`#FFFFFF`
- 字号：`13px #102A43`

### 状态

| 状态 | 边框 | 背景 |
|------|------|------|
| 默认 | `#D9E1E8` | `#FFFFFF` |
| 悬停 | `#B7C3CF` | `#FFFFFF` |
| 聚焦 | `#0F3D5E`（2px） | `#FFFFFF` |
| 错误 | `#C2410C` | `#FEF2F2` |
| 禁用 | `#D9E1E8` | `#F0F4F8` |

### 附属元素

- 前缀图标：`16px`，颜色 `#7B8794`，左侧 `12px` 内边距
- 后缀图标/清除按钮：右侧 `8px`
- 错误提示：输入框下方 `4px`，`12px #C2410C`

---

## 四、下拉选择（Select / Dropdown）

- 触发器样式同 Input，右侧附 `ChevronDown` 图标
- 下拉面板：`--shadow-lg`，圆角 `--radius-md`，背景 `#FFFFFF`
- 选项高度：`32px`，内边距 `0 12px`
- 选项悬停：背景 `#F0F4F8`
- 选中项：背景 `#EBF4FF`，文字 `#0F3D5E`，右侧 `Check` 图标
- 最大高度：`240px`，超出滚动

---

## 五、表格（Table）

### 规格

- 表头：`12px 500 #7B8794`，大写，背景 `#F5F7FA`，高度 `36px`
- 数据行：高度 `44px`（标准）/ `52px`（含副文本）
- 斑马纹：奇数行 `#FFFFFF`，偶数行 `#F5F7FA`（可选，默认关闭）
- 悬停行：背景 `#F0F4F8`，`150ms`
- 选中行：背景 `#EBF4FF`，左侧 `3px solid #0F3D5E`
- 单元格内边距：`0 16px`
- 底部边框：`1px solid #D9E1E8`

### 列类型

| 列类型 | 对齐 | 字体 | 说明 |
|--------|------|------|------|
| 文本 | 左 | `--font-ui` | 名称、描述 |
| 数字/分数 | 右 | `--font-mono` | 评分、数量、大小 |
| 状态 | 左 | — | 使用状态徽标 |
| 时间 | 右 | `--font-mono` | 相对时间（悬停显示绝对时间） |
| 操作 | 右 | — | Ghost 按钮，悬停行才显示 |

### 空状态

- 居中显示，图标 `48px #D9E1E8`，标题 `16px #486581`，描述 `14px #7B8794`
- 可选操作按钮（Secondary 变体）

### 分页

- 位置：表格底部右侧
- 显示：`共 N 条` + 页码导航 + 每页条数选择
- 字号：`13px #7B8794`

---

## 六、卡片（Card）

### 基础规格

- 背景：`#FFFFFF`
- 边框：`1px solid #D9E1E8`
- 圆角：`--radius-md`（12px）
- 阴影：`--shadow-none`（默认）
- 悬停：边框变 `#B7C3CF`，阴影变 `--shadow-md`，`150ms`
- 内边距：`20px`（标准）/ `16px`（紧凑）

### 卡片头部

```
┌─────────────────────────────────────────┐
│ 图标（可选）  标题（16px 600）           │
│              副标题（13px #7B8794）      │
│                              操作按钮区  │
└─────────────────────────────────────────┘
```

- 头部与内容区用 `1px solid #D9E1E8` 分隔（可选）

### 指标卡片（Metric Card）

```
┌──────────────────────────┐
│ 指标名称（12px #7B8794） │
│ 数值（40px 700 mono）    │
│ 趋势标签（12px）         │
└──────────────────────────┘
```

- 数值字体：`--font-mono`，颜色 `#102A43`
- 趋势上升：`#1F9D55` + `TrendingUp` 图标
- 趋势下降：`#C2410C` + `TrendingDown` 图标

---

## 七、进度条（Progress）

### 流水线进度条

- 高度：`6px`，圆角 `3px`
- 背景：`#D9E1E8`
- 填充：按阶段色（见 Design Tokens 流水线阶段色）
- 动画：`width` 过渡 `250ms ease-out`

### 步骤进度（Step Progress）

用于流水线 10 阶段展示：

```
● ─── ● ─── ◐ ─── ○ ─── ○
完成  完成  进行中  待处理  待处理
```

- 完成节点：`#1F9D55` 实心圆 + `Check` 图标
- 进行中：`#0F9F8C` 半圆 + 脉冲动画
- 待处理：`#D9E1E8` 空心圆
- 失败：`#C2410C` + `X` 图标
- 连接线：`2px`，完成段 `#1F9D55`，未完成段 `#D9E1E8`

---

## 八、标签页（Tabs）

### 规格

- 高度：`36px`
- 底部边框：`2px solid #D9E1E8`（容器）
- 激活标签：底部 `2px solid #0F3D5E`，文字 `#102A43 500`
- 非激活：文字 `#7B8794 400`，悬停 `#486581`，`150ms`
- 内边距：`0 16px`

### 变体

- **Line**（默认）：底部线条指示器
- **Pill**：胶囊背景，用于筛选器场景，背景 `#EBF4FF`

---

## 九、弹窗（Modal / Dialog）

### 规格

- 遮罩：`rgba(16,42,67,0.4)`
- 面板：`#FFFFFF`，圆角 `--radius-lg`，阴影 `--shadow-lg`
- 宽度：`480px`（sm）/ `640px`（md）/ `800px`（lg）
- 内边距：`24px`
- 动画：`scale(0.96) → scale(1)` + `opacity 0→1`，`200ms ease-out`

### 结构

```
┌──────────────────────────────────────┐
│ 标题（20px 600）              ✕ 关闭 │
├──────────────────────────────────────┤
│                                      │
│  内容区（14px #486581）              │
│                                      │
├──────────────────────────────────────┤
│                  取消    确认（主操作）│
└──────────────────────────────────────┘
```

- 危险操作弹窗：确认按钮用 Danger 变体，标题前加 `AlertTriangle` 图标

---

## 十、提示（Toast / Notification）

### Toast

- 位置：右下角，距边 `24px`
- 宽度：`320px`
- 圆角：`--radius-md`，阴影 `--shadow-lg`
- 自动消失：`4000ms`（info/success）/ `6000ms`（warning/danger）
- 动画：从右侧滑入 `translateX(100%) → 0`，`200ms ease-out`

| 类型 | 左侧色条 | 图标 |
|------|----------|------|
| success | `#1F9D55` | `CheckCircle` |
| warning | `#D97706` | `AlertTriangle` |
| danger | `#C2410C` | `XCircle` |
| info | `#2563EB` | `Info` |

### 内联提示（Alert）

- 圆角 `--radius-sm`，内边距 `12px 16px`
- 左侧 `4px` 色条
- 图标 `16px` + 标题（`13px 500`）+ 描述（`13px 400`，可选）

---

## 十一、骨架屏（Skeleton）

- 颜色：`#E8EDF2`
- 动画：`shimmer`（从左到右渐变扫描），`1.5s` 循环
- 圆角：与目标元素一致
- 使用场景：首次加载、数据刷新时替代真实内容

---

## 十二、空状态（Empty State）

### 规格

```
        [图标 48px #D9E1E8]
        
    标题（16px 600 #486581）
    描述（14px #7B8794，可选）
    
    [操作按钮（可选）]
```

- 容器：居中，`padding: 48px 24px`
- 图标：Lucide，`48px`，颜色 `#D9E1E8`

### 场景变体

| 场景 | 图标 | 标题 |
|------|------|------|
| 无数据 | `Database` | 暂无数据 |
| 搜索无结果 | `SearchX` | 未找到匹配结果 |
| 无知识库 | `BookOpen` | 还没有知识库 |
| 无文档 | `FileText` | 还没有文档 |
| 加载失败 | `AlertCircle` | 加载失败 |

---

## 十三、代码/文本块（Code Block）

- 背景：`#F5F7FA`
- 边框：`1px solid #D9E1E8`
- 圆角：`--radius-sm`
- 内边距：`12px 16px`
- 字体：`--font-mono`，`12px`，颜色 `#102A43`
- 右上角：复制按钮（Ghost sm，悬停显示）

---

## 十四、工具提示（Tooltip）

- 背景：`#102A43`
- 文字：`#FFFFFF`，`12px`
- 圆角：`--radius-xs`（4px）
- 内边距：`4px 8px`
- 延迟显示：`300ms`
- 最大宽度：`240px`，超出换行
