# 文档归档

本目录存放不再作为当前主入口、但仍有历史参考价值的文档。

## 目录说明

- `agent/`：Claude / Codex 迁移期间的历史代理配置与角色映射。
- `architecture/`：早期问题定位和历史架构分析。
- `legacy-chain/`：早期链路入口迁移说明与旧入口 stub。
- `performance-discussions/`：早期性能讨论、实验记录和专题过程材料。
- `plans/`：已被当前 `.planning/**` 或 `docs/` 结论吸收的历史计划。

## 归档规则

- 当前稳定链路说明进入 `docs/pipeline/`。
- 当前规则边界进入 `docs/rule/`。
- 当前架构口径进入 `docs/architecture/`。
- 当前性能结论进入 `docs/performance-optimizations/`。
- 历史分析、旧入口、旧阶段材料和迁移说明进入 `docs/archive/` 或 `.planning/milestones/`。

归档文档保留原始背景，不强行改写为当前实现事实；如需引用当前事实，应链接到对应当前文档。
