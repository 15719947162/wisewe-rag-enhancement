## 一、总览仪表盘（`/`）

### 布局

```
┌─────────────────────────────────────────────────────────┐
│ 页面标题：系统总览          右侧：刷新按钮 + 最后更新时间 │
├──────────────────────────────┬──────────────────────────┤
│  系统健康卡片（7列）         │  告警面板（5列）          │
│  ─────────────────────────── │  ─────────────────────── │
│  4个指标卡片（2×2）          │  告警列表（最近5条）      │
├──────────────────────────────┴──────────────────────────┤
│  知识库概览表格（全宽）                                   │
├─────────────────────────────────────────────────────────┤
│  最近入库任务（全宽，最近10条）                           │
└─────────────────────────────────────────────────────────┘
```

### 系统健康卡片

4个指标卡片，2×2 网格：

| 指标 | 图标 | 数值字体 |
|------|------|----------|
| 知识库总数 | `Database` | `--font-mono` |
| 文档总数 | `FileText` | `--font-mono` |
| 向量总数 | `Layers` | `--font-mono` |
| 平均质量分 | `Star` | `--font-mono` |

- 数值：`40px 700 #102A43`，`--font-mono`
- 指标名：`12px 500 #7B8794`，大写

### 告警面板

- 标题：`告警` + 未读数字徽标
- 每条告警：左侧色条（danger/warning）+ 告警描述 + 时间
- 无告警时：显示 `CheckCircle` 空状态，文字"系统运行正常"

### 知识库概览表格

| 列 | 宽度 | 类型 |
|----|------|------|
| 知识库名称 | flex | 文本 + 点击跳转 |
| 文档数 | 80px | 数字 |
| 向量数 | 100px | 数字 |
| 平均质量分 | 100px | 数字（mono） |
| 最后更新 | 120px | 时间 |
| 状态 | 80px | 状态徽标 |

### 最近入库任务

- 显示最近 10 条，含：文件名、知识库、阶段进度条（缩略）、状态、时间
- 点击行跳转到文档入库页对应任务

## 三、文档入库（`/ingestion`）

### 三栏布局

```
┌──────────────────────────────────────────────────────────────┐
│ 文档入库                                        上传文档      │
├────────────┬──────────────────────────┬─────────────────────┤
│ 任务队列   │  主工作区（流水线详情）  │  阶段详情侧栏       │
│ 240px      │  flex-1                  │  260px              │
└────────────┴──────────────────────────┴─────────────────────┘
```

### 任务队列（左栏）

- 每个任务项：文件名（截断）+ 状态徽标 + 进度百分比
- 高度：`56px`
- 激活任务：左侧 `3px solid #0F3D5E`，背景 `#EBF4FF`
- 分组：进行中 / 排队中 / 已完成 / 失败

### 主工作区（中栏）

**流水线步骤进度**（10阶段）：

```
① 上传OSS → ② 提交解析 → ③ 轮询解析 → ④ 下载结果
→ ⑤ 规则清洗 → ⑥ LLM清洗 → ⑦ 切片 → ⑧ 质量门控
→ ⑨ 向量化 → ⑩ 写入pgvector
```

- 步骤节点：圆形 `32px`，状态色填充
- 连接线：`2px`，完成段品牌色，未完成段 `#D9E1E8`
- 当前阶段：节点脉冲动画 + 下方显示实时日志（最近3条）
- 失败阶段：红色节点 + 错误摘要（点击展开详情）

**阶段耗时统计**（完成后显示）：

- 横向条形图，每阶段一行
- 颜色对应阶段色（见 Design Tokens）

### 阶段详情侧栏（右栏）

- 标题：当前选中阶段名称
- 内容：
  - 阶段状态徽标
  - 开始/结束时间
  - 耗时
  - 输入/输出数量
  - 日志（可展开，代码块样式）
  - 错误详情（失败时显示，红色内联提示）

### 上传文档弹窗

- 拖拽区域：`200px` 高，虚线边框 `#D9E1E8`，悬停变品牌色
- 支持格式：PDF（显示在拖拽区下方）
- 知识库选择：下拉选择器
- 切片策略选择：Radio 组（6种策略）
- 高级参数：折叠面板（默认收起）

## 七、通用交互状态规范

### 加载状态

| 场景 | 处理方式 |
|------|----------|
| 页面首次加载 | 骨架屏（Skeleton） |
| 数据刷新 | 表格/列表区域 opacity 0.5 + 顶部进度条 |
| 按钮操作 | 按钮内 Loader 图标旋转，禁用点击 |
| 流水线实时更新 | 步骤节点脉冲 + 日志滚动 |

### 错误状态

| 场景 | 处理方式 |
|------|----------|
| 网络错误 | 全局 Toast（danger）+ 重试按钮 |
| 表单验证 | 字段下方内联错误文字 |
| 流水线失败 | 阶段节点变红 + 错误摘要 + 查看日志链接 |
| 空数据 | 空状态组件（见组件规范） |

### 确认操作

- 删除知识库：弹窗确认，需输入知识库名称
- 删除文档：弹窗确认（简单确认即可）
- 清空数据：弹窗确认，Danger 按钮

### 实时更新

- 流水线进行中：每 `2s` 轮询，步骤状态平滑过渡
- 仪表盘指标：每 `30s` 静默刷新（不触发骨架屏）
- 告警：WebSocket 推送（降级为轮询）
